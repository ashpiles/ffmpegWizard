# FFMPEG WIZARD

### A tool for saving and creating ffmpeg commands with drag and drop functionality.

The application aims to help users get away from the command line while still utilizing the full power of ffmpeg all from a user friendly GUI.

Simply drag and drop commands or flags to create the desired ffmpeg command to create a personal library of commands you use regularly.

## How to Use
- Run application from command line for access to error messages
- Drag a desired command to the center of the application
- Edit flag values by double clicking or by dragging and dropping
- Hit Run Command
- That's it!

#### Warning
At the moment if you try and set the output file name as an existing file the application will get stuck and need to be restarted
